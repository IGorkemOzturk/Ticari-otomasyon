# Ticari-otomasyon
Kurumsal firmaların hem kendi içerisindeki iletişimi hem de dışarıdaki firmalarla iletişim kurabilmesini sağlayan bir proje. Diğer yandan firmalar kendi içlerinde envanter kontrolü, satış kayıtları ve kargo takipleri gibi bir çok kayıtı tutup üzerinde çalışabilirler.
